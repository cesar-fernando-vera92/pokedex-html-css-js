# 🗺️ Pokedex 

🚀 **DEMO** (https://cesar-fernando-vera92.github.io/pokedex-html-css-js/)

---

## ✨ Características

- **Consumo de API:** Conexión directa con la PokéAPI de forma asíncrona (`fetch`).
- **Búsqueda Dinámica:** Permite a los usuarios buscar a sus Pokémon favoritos por su nombre o su número de ID.
- **Navegación:** Botones de navegación (Anterior/Siguiente) para explorar la lista secuencialmente.
- **Diseño Responsivo:** Adaptado para visualizarse correctamente tanto en dispositivos móviles como en ordenadores.
